# Компрометация учетной записи

## Описание
Компрометация аккаунта означает, что злоумышленник получил доступ
к учетной записи пользователя.

## Типовой сценарий атаки
Злоумышленник входит в аккаунт с неизвестного устройства.

## Признаки
- Входы с неизвестных IP-адресов
- Изменение настроек аккаунта

## План реагирования

### 1. Немедленные действия
- Сменить пароль
- Завершить все активные сессии

### 2. Анализ
- Проверить журнал входов
- Проверить изменения данных

### 3. Восстановление
- Вернуть корректные настройки
- Уведомить пользователя

### 4. Профилактика
- Использование двухфакторной аутентификации
