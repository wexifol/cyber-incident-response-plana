# Компрометация учетной записи

## Описание
Злоумышленник получил доступ к аккаунту пользователя.

## Признаки
- Входы с неизвестных устройств
- Изменение настроек без ведома пользователя

## План реагирования

### 1. Немедленные действия
- Сменить пароль
- Завершить все сессии

### 2. Анализ
- Проверить журнал входов
- Проверить действия в аккаунте

### 3. Профилактика
- Использовать двухфакторную аутентификацию
