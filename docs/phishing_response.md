# Реагирование на фишинговую атаку

## Признаки атаки
- Подозрительное письмо
- Запрос логина и пароля
- Странная ссылка

## Действия

### 1. Немедленно
- Сменить пароль
- Заблокировать аккаунт
- Сообщить администратору

### 2. Проверка
- Проверить входы в аккаунт
- Проверить другие письма

### 3. Профилактика
- Не открывать подозрительные письма
- Использовать сложные пароли
