# Фишинговая атака

## Описание
Фишинг — это вид социальной инженерии, при котором злоумышленник
пытается обманом получить логин, пароль или другие конфиденциальные
данные пользователя.

## Типовой сценарий атаки
Пользователь получает письмо, замаскированное под сообщение от
банка или сервиса, и переходит по вредоносной ссылке.

## Признаки атаки
- Срочные требования («аккаунт будет заблокирован»)
- Подозрительный адрес отправителя
- Ссылки, ведущие на сторонние сайты

## План реагирования

### 1. Немедленные действия
- Немедленно сменить пароль
- Завершить все активные сессии
- Заблокировать учетную запись при необходимости

### 2. Анализ инцидента
- Проверить журнал входов
- Определить, были ли несанкционированные действия

### 3. Устранение последствий
- Удалить фишинговые письма
- Предупредить других пользователей

### 4. Профилактика
- Обучение пользователей
- Использование двухфакторной аутентификации
