# Фишинговая атака

## Описание
Фишинг — это атака, при которой злоумышленник пытается получить
доступ к учетным данным пользователя с помощью поддельных писем
или сайтов.

## Признаки атаки
- Письмо с просьбой срочно ввести пароль
- Подозрительная ссылка
- Ошибки в тексте письма

## План реагирования

### 1. Немедленные действия
- Сменить пароль пользователя
- Завершить все активные сессии
- Заблокировать учетную запись при необходимости

### 2. Анализ
- Проверить историю входов
- Проверить, были ли подозрительные действия

### 3. Профилактика
- Обучить пользователя
- Использовать сложные пароли
- Включить двухфакторную аутентификацию
