# DDoS-атака

## Описание
DDoS-атака направлена на отказ в обслуживании сервиса за счет
массового потока запросов.

## Типовой сценарий атаки
Сайт или сервер становится недоступным из-за перегрузки.

## Признаки атаки
- Резкое увеличение трафика
- Медленная или невозможная загрузка сайта

## План реагирования

### 1. Немедленные действия
- Ограничить входящий трафик
- Уведомить хостинг-провайдера

### 2. Анализ
- Определить источники трафика
- Выявить подозрительные IP-адреса

### 3. Устранение
- Блокировка IP-адресов
- Включение защиты от DDoS

### 4. Профилактика
- Использование CDN
- Настройка лимитов запросов
