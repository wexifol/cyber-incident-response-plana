# DDoS-атака

## Описание
DDoS — это распределённая атака типа отказ в обслуживании, когда
злоумышленники перегружают сервер запросами, делая сервис недоступным. :contentReference[oaicite:2]{index=2}

## Признаки
- Резкий рост входящего трафика  
- Сайт перестал загружаться

## План реагирования

### 1. Немедленные действия
- Ограничить входящий трафик  
- Уведомить хостинг-провайдера

### 2. Анализ
- Определить источники трафика

### 3. Устранение
- Заблокировать подозрительные IP
- Включить защиту от DDoS

### 4. Профилактика
- Настроить CDN  
- Ограничить лимиты запросов
