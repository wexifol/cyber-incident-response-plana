# DDoS-атака

## Описание
DDoS-атака направлена на перегрузку сервиса большим количеством
запросов, из-за чего он становится недоступным.

## Признаки
- Сайт не открывается
- Резкий рост трафика

## План реагирования

### 1. Немедленные действия
- Ограничить входящий трафик
- Сообщить хостинг-провайдеру

### 2. Анализ
- Проверить IP-адреса источников атак

### 3. Профилактика
- Использовать защиту от DDoS
- Настроить лимиты запросов
