# Кризисный план действий при кибератаках

## Описание проекта
Данный проект представляет собой набор пошаговых инструкций
для реагирования на различные виды кибератак. Проект предназначен
для использования в учебных целях и демонстрирует базовые принципы
инцидент-менеджмента в области информационной безопасности.

## Решаемая проблема
Во время кибератак пользователи и организации часто не понимают,
какие действия необходимо выполнять в первую очередь. Это приводит
к увеличению ущерба и потере данных.

Проект предлагает готовые сценарии реагирования, которые можно
повторить на практике.

## Рассматриваемые сценарии атак
- Фишинговая атака
- Заражение вредоносным ПО
- DDoS-атака
- Компрометация учетной записи

## Общая логика реагирования
1. Обнаружение инцидента
2. Оценка ситуации
3. Немедленные действия
4. Устранение последствий
5. Профилактика

## Используемые инструменты
Описание используемых инструментов находится в папке tools.

## Проверка работоспособности
Решения протестированы с помощью сценарного тестирования
(tabletop testing). Результаты описаны в папке tests.
