# Используемые инструменты

## Журналы входа
Используются для проверки подозрительных входов в учетные записи.

## Антивирус
Применяется для поиска и удаления вредоносного ПО.

## Whois
Используется для проверки подозрительных доменов и IP-адресов.

## Ограничение сессий
Позволяет завершить активные подключения злоумышленника.
