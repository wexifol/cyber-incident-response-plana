# Используемые инструменты

## Журналы входа
Используются для анализа активности учетных записей и выявления
подозрительных входов.

## Антивирусное ПО
Применяется для обнаружения и удаления вредоносных программ.

## Whois и анализ IP
Позволяет определить принадлежность доменов и IP-адресов,
используемых в атаках.

## Управление сессиями
Используется для завершения активных подключений злоумышленника.
